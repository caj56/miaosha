# miaosha

> 从零开始基于Golang实现秒杀系统，技术栈：Golang、Gin、RabbitMQ、Redis、MySQL等，主要功能包括用户登录注册、分布式 Session、秒杀、异步下单、防刷限流等，采用Jenkins+Docker部署，文档逐渐完善，欢迎大家一起学习交流。
